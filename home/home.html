O que é Micro-Saas?

Micro-SaaS é um serviço de software online que resolve um problema bem específico para um grupo pequeno de pessoas. Ele é criado por uma ou poucas pessoas e funciona de maneira simples, como uma ferramenta extra que melhora algo que você já usa, como um complemento para um programa. É como se fosse uma pequena solução feita sob medida para uma necessidade específica, disponível pela internet.
